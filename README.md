# Cloud Complaint Management System

A cloud-based complaint management system built using Flask and AWS.

## Live Demo
http://complaint-env.eba-mfrqfvxx.eu-north-1.elasticbeanstalk.com

## Features
- User Registration & Login
- Submit Complaint with Image
- View Complaint Status
- Admin Dashboard to Manage Complaints

## Tech Stack
- Flask (Backend)
- AWS Elastic Beanstalk (Deployment)
- Amazon RDS (Database)
- Amazon S3 (Image Storage)

## How to Run
1. Install requirements
2. Run application.py
3. Open browser at localhost:5000
